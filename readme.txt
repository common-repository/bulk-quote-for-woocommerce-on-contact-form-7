== Description ==
Add Woocommerce bulk quote fields to the popular Contact Form 7 plugin.
Select product category from contact form 7 tag woo bulk quote.
You can choose how many columns to display on form.

== Installation ==
The Contact Form 7 plugin and WooCommerce plugin must be installed and activated for the WooCommerce bulk quote plugin to work.
